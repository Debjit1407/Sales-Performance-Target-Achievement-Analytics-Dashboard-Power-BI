# Sales-Performance-Target-Achievement-Analytics-Dashboard-Power-BI📊 Power BI Sales Dashboard
📌 Overview

This project is a Power BI solution built to analyze sales performance, profit, returns, delivery delays, and target achievement using multiple yearly datasets.

📂 Data Sources
order_2015 to order_2018
Returns
People
Target
🔄 Data Preparation
Combined yearly order files into FactSales
Cleaned Quantity (text → number)
Converted dates to proper format
Removed “K” from Target values
🧩 Data Model
FactSales (main table)
DateTable (DAX calendar)
Relationships with People, Returns, and Target tables
📐 Key Measures
Sales, Profit, Orders, Quantity
Return % and Delay %
Target Achievement & Gap
YOY Growth (Sales & Profit)
📊 Dashboard Pages
1. Executive Summary

KPI cards, sales trend, region and category analysis

2. Manager Performance

Sales vs Target with conditional formatting

3. Category Analysis

Target vs Sales heatmap

4. Drillthrough

Detailed manager-level performance

🎯 Outcome

An interactive dashboard providing insights into sales trends, performance gaps, and business risks for better decision-making.
